Arrays com PHP





No PHP a principal finalidade da estrutura Array é guardar informações de modo ordenado através de índices, uma linha para cada informação, um exemplo comum disso é guardar notas de alunos de uma sala por exemplo ou então nomes, id, valores para o sistema etc.

Em muitos programas os Arrays são utilizados, pois com sua utilização é simplificado o trabalho com manipulação de dados, substituindo o uso de várias variáveis primitivas em um só código.

Por exemplo, em um cenário hipotético no qual precisamos manipular as notas de 30 alunos em uma sala ou até mesmo da escola inteira em um banco de dados, onde para cada aluno tivesse os seguintes campos: nome, sala, nota 1, nota 2, nota final e tivéssemos que criar variáveis para armazenar todos esses dados de todos os alunos, seria bastante trabalhoso e inviável, pois além de criar uma enorme quantidade de variáveis no código, ficaria extremamente desorganizado para quem além de você fosse ler e manipular depois.

Então qual seria a solução do problema? Simples, poderíamos resolver a situação utilizando um Array multidimensional, onde teríamos apenas o ponto de acesso para os dados, que no caso seria o Array criado, e dentro do Array teríamos outros Arrays que guardariam as notas dos alunos obtidos do banco.

Portanto os Arrays nada mais é do que uma estrutura de dados que armazena os dados de forma contínua através de índices.
