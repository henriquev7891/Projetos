Estruturas de Controle e Estruturas de Repetição no PHP





No PHP e na programação em geral as estruturas de controle e as estruturas de repetição são de grande importância além de serem constantemente utilizadas na maioria dos programas.

A Estrutura de Controle, também conhecida como estrutura condicional é utilizado a função/comando if (SE) , else (SE NÃO) ou switch que é executado testes em um bloco de código com uma determinada condição.

Enquanto na Estrutura de Repetição é utilizado um laço de repetição podendo ser a função/comando while, do while ou for, que no caso nas estruturas de repetição um bloco de código é repetido várias vezes até o final da instrução ou até uma determinada quantia de vezes, que normalmente é definido através de um contador.

Nas Estruturas de Repetição também é possível utilizar dentro delas uma Estrutura de Controle, fazendo com que uma determinada parte do código possa repetir várias vezes a partir de uma determinada condição.
