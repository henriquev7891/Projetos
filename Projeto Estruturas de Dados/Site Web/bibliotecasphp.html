Bibliotecas no PHP



No PHP existem algumas bibliotecas que auxiliam a desenvolver os programas e ajudam na solução de problemas comuns e complexos dentro da programação, uma delas é a Biblioteca SPL ou PHP-SPL (SplDoublyLinkedList).

Esta Biblioteca fornece soluções que facilitam alguns problemas comuns encontrados na Estrutura de Dados no PHP, além de oferecer uma coleção de interfaces, classes e funções para trabalhar com Estruturas de Dados no PHP.
