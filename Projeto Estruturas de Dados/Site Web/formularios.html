Formulários com PHP





Os formulários fazem um papel de extrema importância nos sites e aplicativos em geral, pois nele é possível interagir com o usuário/cliente e, é com ele que se torna possível a obtenção de valores, dados e informações fornecidas pelo usuário, além de validar funções do próprio site ou sistema a partir das informações coletadas.

Segundo a Mozilla, formulários HTML são “um dos principais pontos de interação entre um usuário e um web site ou aplicativo”. Os dados são enviados pelos usuários e coletados pelo servidor web, validando e usando no próprio site. São compostos por um ou mais campos de entrada e podem ser campos de texto, caixa de seleção, botões, checkbox ou radio buttons.


O PHP faz uma bela dupla com os formulários Web, pois ele é utilizado como backend, fazendo com que o formulário feito em HTML5 (exemplo), se torne funcional com o servidor ou localmente, coletando dados ou informações que serão posteriormente utilizadas no próprio site ou para validar alguma funcionalidade específica no sistema por exemplo uma tela de Login.

Existem diversas maneiras do PHP coletar informações ou valores através dos formulários, normalmente é utilizado para isso os métodos/funções http $_GET e $_POST .

Métodos $_GET e $_POST:

$_GET: Recebe os dados diretamente dos campos do formulário ou através da URL com a possibilidade de alterar os dados pela URL após um ponto de interrogação, exemplo:

www.site.com.br/exemploGET.php?variavel=valor
