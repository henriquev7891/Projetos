Listas (do tipo filas e do tipo pilhas) no PHP



A estrutura Lista no PHP é utilizada em vários casos em programas e sendo normalmente dois tipos, a do tipo pilha que tem como função armazenar elementos em uma ordem específica, que no caso é uma em cima da outra como se fosse um empilhamento de livros e a do tipo fila que é utilizada para a inserção ou armazenamento de dados de forma que, o primeiro elemento inserido também será o primeiro a ser removido diferente da pilha que o primeiro será o último.

A estrutura Lista também conta com o tipo Lista ligada que é um tipo de lista onde os dados armazenados têm uma ligação com um certo valor, que no caso ela conhece o valor que está sendo armazenado em seu interior além de conhecer o elemento posterior a ele e conta com o tipo de lista duplamente ligada que a principal diferença é por ela ser bidirecional.


Listas Ligada: Na estrutura do tipo lista existem os nós onde cada um dos nós conhece o valor que está sendo armazenado em seu interior além de conhecer o elemento posterior a ele: por isso ela é chamada de "lista ligada", pois os nós são amarrados com essa indicação de qual é o próximo nó.

Lista Duplamente Ligada: A grande diferença das listas duplamente ligadas para as listas ligadas é que elas são bidirecionais. Vimos que, naturalmente, não conseguimos "andar para trás" em listas ligadas, pois os nós de uma lista ligada sabem somente quem é o próximo elemento. Nas listas duplamente ligadas, os nós sabem quem é o próximo elemento e também quem é o elemento anterior, o que permite a navegação reversa.

Fila: Fila é uma estrutura de dados bastante utilizada para inserção ou armazenamento de informações ou dados de forma que, o primeiro elemento inserido também será o primeiro a ser removido.

Tendo como restrição/regra:

A restrição da Estrutura de Dado Fila é que o primeiro elemento inserido também será o primeiro a ser removido, ou seja, é adicionado itens e quando for remover no final começa a remover os primeiros.

Pilhas: LIFO ou PEPS, FIFO ou UEPS.

Pilha LIFO ou UEPS: A estrutura do tipo Pilha LIFO (Last in First Out) ou UEPS (Último que Entra Primeiro que Sai), apresenta o seguinte critério: O primeiro elemento a ser retirado é o último que tiver sido inserido.

Pilha FIFO ou PEPS: A estrutura do tipo PILHA FIFO (First in First Out) ou PEPS (Primeiro que Entra Primeiro que Sai), apresenta o seguinte critério: O primeiro elemento a ser retirado é o primeiro que tiver sido inserido.

Tendo como restrição/regra:

A restrição na Pilha é que o último elemento inserido será o primeiro a ser retirado. Portando a regra de funcionamento na pilha é que os elementos só podem ser inseridos (empilhados) sobre o último elemento já existente.
