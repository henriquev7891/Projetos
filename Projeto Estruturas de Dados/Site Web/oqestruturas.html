O que é Estrutura de Dados?




O termo “Estruturas de Dados” em diversos contextos e assuntos associados a programação, refere-se a “processamento de dados”.

A Estrutura de Dados na programação em geral é bastante utilizada para o processamento de dados ou informações de maneira eficiente.

Embora seja muito associada a manipulação ou organização de dados, existe procedimentos/casos em que é possível utilizar sem ter que manipular nenhum dado, entretanto é considerado “pouco prático”.

Portanto a Estrutura de Dados é um ramo da programação que estuda diversas maneiras de organização e manipulação de dados através de programas, nela é possível manipular ou organizar arquivos, dados, valores, etc de forma eficiente além de prover soluções para problemas comuns e complexos encontrados no desenvolvimento de programas, sites, aplicativos e afins.


Principais Estruturas de Dados:


Arrays, Lista, Pilha, Fila, Árvore, Tabela Hash, Grafos


Arrays: Uma estrutura que armazena os dados de forma contínua através de índices.

Listas: Uma estrutura que armazena/organiza elementos ou dados em uma ordem específica.

Listas Ligada: Na estrutura do tipo lista existem os nós onde cada um dos nós conhece o valor que está sendo armazenado em seu interior além de conhecer o elemento posterior a ele: por isso ela é chamada de "lista ligada", pois os nós são amarrados com essa indicação de qual é o próximo nó.

Lista Duplamente Ligada: A grande diferença das listas duplamente ligadas para as listas ligadas é que elas são bidirecionais. Vimos que, naturalmente, não conseguimos "andar para trás" em listas ligadas, pois os nós de uma lista ligada sabem somente quem é o próximo elemento. Nas listas duplamente ligadas, os nós sabem quem é o próximo elemento e também quem é o elemento anterior, o que permite a navegação reversa.

Pilhas: A estrutura pilha serve para armazenar dados em uma ordem específica na pilha os últimos são os primeiros, um exemplo comum de pilha é o empilhamento de livros em que o último livro empilhado será o primeiro a ser removido.

Existem dois tipos de Pilhas: LIFO ou PEPS, FIFO ou UEPS.

Pilha LIFO ou UEPS: A estrutura do tipo Pilha LIFO (Last in First Out) ou UEPS (Último que Entra Primeiro que Sai), apresenta o seguinte critério: O primeiro elemento a ser retirado é o último que tiver sido inserido.

Pilha FIFO ou PEPS: A estrutura do tipo PILHA FIFO (First in First Out) ou PEPS (Primeiro que Entra Primeiro que Sai), apresenta o seguinte critério: O primeiro elemento a ser retirado é o primeiro que tiver sido inserido.

Fila: Fila é uma estrutura de dados bastante utilizada para inserção ou armazenamento de informações ou dados de forma que, o primeiro elemento inserido também será o primeiro a ser removido.

Árvores: É uma estrutura de dados que organiza seus elementos de forma hierárquica, onde existe um elemento que fica no topo da árvore, chamado de raiz e existem os elementos subordinados a ele, que são chamados de nós ou folhas.
