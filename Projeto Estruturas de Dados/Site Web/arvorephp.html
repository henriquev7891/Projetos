Árvores no PHP





A estrutura de dados Árvore é uma estrutura não-linear que permite modelar ramificações e escolhas além de permitir consultas rápidas através de constantes pesquisas de dados a partir de uma entrada.

Uma árvore contém um conjunto finito de um ou mais elementos, normalmente chamados de nós, crescendo de cima para baixo começando pela RAIZ que é o ponto de entrada para a estrutura.

É comum a estrutura árvore ser separada por seis elementos sendo eles:


Raiz que consiste no primeiro elemento da árvore;

Nó que representa um registro contido na árvore;

Elemento Filho que sucede um nó;

Elemento Pai que precede um nó;

Elemento folha que consiste em elementos que não possui filhos;

Elemento interno que consiste em um nó que possui filhos;


Também é comum a estrutura ser separada por sub-árvores com cada sub-árvores tendo seu nó raiz.
