Manipulação de Arquivos com PHP





Um arquivo nada mais é do que um conjunto de documentos, fotos, códigos etc. Podendo ser em diversos formatos, sendo normalmente criados com um propósito.

No PHP é possível gerar, manipular ou organizar arquivos de forma eficiente utilizando algumas funções como fopen, fget, fwrite etc.


Algumas Funções para manipular arquivos em PHP:



fopen() – ‘Abre’ o arquivo para que possa ser manipulado.

fgets() – Pega uma linha do arquivo até o máximo de 1024bytes.

feof() – Durante a leitura de um arquivo, avisa se chegou ao final.

file_get_contents() – Pega todo conteúdo do arquivo aberto como uma string.

ftruncate() – Reduz o tamanho do arquivo. Usado para apagar seu conteúdo.

fwrite() – Escreve no arquivo.

unlink() – Apaga o arquivo indicado.


Exemplo de Manipulação de Arquivo em PHP

<?php
//Criando o Arquivo com uma das funções de Manipulação "fopen".

$arquivo = fopen('EstruturaDados.txt','w');
//Verificando se o arquivo foi criado.

if ($arquivo == false) die('Não foi possível criar o arquivo.');
//Inserindo um contéudo dentro do .txt.

$texto = “Exemplo de Estrutura de Dados”;
fwrite($arquivo, $ texto);

//Fechando o Arquivo .txt após escrever nele.

fclose($arquivo);
?>
